# leftpad.cpp
Pads a string with whitespace

## How to install
0. Clone this repository
1. Wait
2. ...
3. ...
4. ...
5. Think about what you've done

## How to use
```cpp
#include <string>
#include "leftpad.hpp"

int main()
{
	leftpad::leftpad("hello", 6);  	// " hello"
	leftpad::leftpad("hello", 4);  	// "hello"
	leftpad::leftpad("hi", 8);	// "      hi"
}
```
